## Key Features

- **Dual Interface Modes**
  
  - Full-featured toolbar (File, Play, View, Audio, Video, Subtitles, Tools, Effects & Filters, Help)

- **System-Tray Control**
  Keep playback running after closing the main window; right-click the tray icon for instant access to Play, Stop, Next, Previous, Volume, Mute, Open Media or Exit.

- **Built-in Library**
  ‚Äì **Favorites** & **Playlists** support tabs with quick ‚ÄúPlay All‚Äù action
  ‚Äì **Load / Clear Thumbnails** for quick visual browsing

- **10-Band Audio Equalizer**
  31 Hz ‚Äì 16 kHz sliders + pre-amp, max-volume boost, custom & preset storage.

- **Video Effects & Filters**
  
  - Image Adjust: Hue, Brightness, Contrast, Saturation, Gamma  
  - JavaFX Filters: Bloom, Box/Gaussian/Motion Blur, Glow, Inner Shadow, SepiaTone, Reflection (adjustable threshold)  
  - Lighting: Diffuse & Specular constants, surface scale, azimuth/elevation, color picker  
  - Geometry: Rotation, orientation, independent X/Y scale, rotation-axis controls  
  - Crop: pixel-precise Top/Bottom & Left/Right cropping

- **Media Information Panel**
  Edit and save metadata (Title, Artist, Album, Year, Genre, Language, Publisher, Copyright, Comments, Encoded By) plus codec details for every video/audio track.

- **Real-Time Statistics**
  Live counters for decoded/played/lost audio blocks & video frames, current FPS, input bitrate and data size; export stats to text file.

- **Live Caption (Vosk 0.3.50)**
  On-the-fly speech-to-text overlay under the video; debug mode, start/stop & reset buttons. (This feature is not yet fully workable.) 

## Usage

Drag-and-drop or open any common audio/video file, choose your control style, apply filters on the fly, and enjoy gap-free playback with full tray integration.
